Workout Tracker (Google Sheets API Integration)

An automation tool built using python to track daily workouts. It leverages Natural Language Processing to process normal human sentences (e.g., "swam for 45 mins and walked 2 miles") into broken down data metrics, logging them instantly into a targeted Google Sheet.

This is part of Day 38 of Angela Yu's *100 Days of Code: The Complete Python Pro Bootcamp*.
